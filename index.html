Generate Token API

The Generate Token API is designed to authenticate clients and provide them with a secure access token. This token is then used to access other protected APIs within the system.

API URL

https://[domain-sub-domain]:[port]/push-task-without-document /

User has to send a POST request with content type application/json using the above URL and below parameters

REQUEST STRUCUTRE

Key Sample Data Constraint (if Any) Description

clientName Parishram Parishram The name of the client organization

clientSecretKey example@2015 example@2015 clientSecretKey example@2015 example@2015 A secret key for authentication, crucial for securing API access.

RESPONSE STRUCTURE

Key Sample Data Description
status True/false Status of request
message Success Status message
data Object All the information inside data
-token example@2015 used for authentication in subsequent API requests.

PUSH TASK API

The PUSH TASK API allows third party or service provider systems to push task information to Parishram platform.

API URL

https://[domain-sub-domain]:[port]/push-task-without-document / User has to send a POST request with content type application/json with Authorization: Bearer

using the above URL and below parameters

Key Sample Data Constraint (if Any) Description
